## üö® Potential Gotchas & Fixes

### Issue 1: Vercel Deployment Not Live
**Symptom**: team-consensus-vault.vercel.app shows 404 or "deployment not found"
**Fix Before Recording**:
```bash
# Re-deploy to Vercel
cd ~/team-consensus-vault
npx vercel --prod
# OR push to GitHub and Vercel auto-deploys
```
**Workaround**: Use localhost if deployment fails (not ideal but workable)

### Issue 2: Wallet Connection Fails
**Symptom**: RainbowKit modal doesn't appear or wallet doesn't connect
**Fix**:
- Check that wallet extension is installed (MetaMask/Rainbow/Coinbase)
- Verify browser permissions for app
- Test in incognito mode to rule out extension conflicts
**Workaround**: Skip wallet connection step, verbally describe it

### Issue 3: Consensus Doesn't Reach 80%
**Symptom**: Query submitted but consensus stays below threshold
**Fix**: This is actually fine ‚Äî shows system working correctly
**Workaround**: Submit different query (try Bitcoin query for higher consensus)
**Demo Adjustment**: "Notice the consensus is only 65% here ‚Äî the system is being cautious and won't auto-trade without strong agreement."

### Issue 4: SSE Stream Errors
**Symptom**: Analyst cards stuck on "Analyzing..." forever
**Fix**:
- Check browser console for errors
- Verify `/api/consensus` endpoint is accessible
- Refresh page and try again
**Workaround**: Use mock data mode (should auto-activate if SSE fails)

### Issue 5: Audio Quality Poor
**Symptom**: Recording sounds muffled, echoing, or has background noise
**Fix**:
- Use external mic (not laptop mic)
- Record in closet or room with soft furnishings (reduces echo)
- Use noise suppression in OBS/Loom settings
**Workaround**: Re-record audio separately and overlay in editing

### Issue 6: Screen Recording Laggy
**Symptom**: Mouse movements are choppy, animations stutter
**Fix**:
- Close all other applications
- Use 30fps instead of 60fps
- Record at 720p instead of 1080p
- Disable hardware acceleration in browser
**Workaround**: Record shorter clips and stitch together

### Issue 7: Run Out of Time
**Symptom**: Video exceeds 5 minutes
**Fix**:
- Cut intro to 20 seconds (reduce story, get to demo faster)
- Shorten tech stack to 30 seconds (list stack, skip details)
- Remove future vision section entirely
**Minimum Viable Video**: Intro (0:20) + Demo (2:30) + Credits (0:10) = 3:00 total
